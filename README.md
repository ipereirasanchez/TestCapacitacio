# Test Capacitació

Aplicación web para realizar tests de preguntas de opción múltiple.

## Características

- **Modo por temas**: Permite elegir temas específicos y número de preguntas.
- **Modo 200 de todo**: Test completo con preguntas de todos los temas.
- Feedback inmediato y corrección.
- Diseño responsive adaptado a móviles y escritorio.

## Uso

Simplemente abre el archivo `index.html` en tu navegador web.
